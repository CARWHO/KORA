   (a) y = sin x / (2x + 1)
   (b) x³ - y³ = 5xy²
   (c) y = 4 cos⁵ (2x)
   
2. (25 pts) The position value of a particle is given by s(t) = t² - 4t⁵ + 4t, where t ≥0 is in seconds and position is in inches.
   (a) Find the particle’s velocity function v(t).
   (b) Determine the particle’s speed at t = 9
   (c) Find the particle’s acceleration function a(t).
   (d) Find all values of t ≥0 for which the particle’s acceleration is equal to 5
   
3. (25 pts)
   (a) Find the equations of the tangent and normal lines to the curve y = x³ - 2x² + x + 10 at x = -6
   (b) Find all values of x on the interval [0, π] for which the curve y = tan x - 4x has a horizontal tangent
   
4. (20 pts)
   (a) Determine f′(x) for the function f(x) = √(x + 1) by using the definition of the derivative.
